# MicroZen

MicroZen es una aplicación mínima de React Native para prácticas de respiración y reflexión personal.

## Funcionalidades
- Sesión de respiración guiada de 60 segundos con animación de un círculo que se expande y contrae.
- Campo de texto para escribir una reflexión personal.
- Guardado de reflexiones con fecha y hora en una lista ordenada cronológicamente.

## Uso
1. Instala las dependencias con `npm install` (requiere Node.js y Expo CLI).
2. Ejecuta `npm start` para iniciar la aplicación con Expo.

El diseño es intencionalmente simple para mantener un enfoque estilo Zen.
